# Games - Up Admin Dashboard Credentials

## Default Admin Account

The system comes with a pre-configured admin account for easy access:

### Login Credentials
- **Email:** `admin@gamesup.com`
- **Password:** `Admin@123`
- **Role:** Super Admin

## Features

- **Auto-Fill:** Click the "Click to auto-fill credentials" button on the login page
- **Full Access:** This account has complete access to all dashboard features
- **Pre-Configured:** Account is automatically created when you first load the application

## Creating Additional Users

To create additional admin users:

1. Click "Don't have an account? Sign up" on the login page
2. Enter your details:
   - Full Name
   - Email Address
   - Password (minimum 6 characters recommended)
3. Click "Create Account"
4. You'll be automatically logged in

## Security Notes

⚠️ **Important:** For production use, please change the default admin password immediately after first login.

## What You Can Do

With admin credentials, you have access to:

- ✅ Dashboard analytics and KPIs
- ✅ Product management (add, edit, delete)
- ✅ Order management and tracking
- ✅ Customer database
- ✅ Outlook accounts management
- ✅ Task and team management
- ✅ Role-based access control
- ✅ System settings
- ✅ All admin-only fields (purchased emails, passwords, product codes)

## Quick Start

1. Open the application
2. On the login page, click "Click to auto-fill credentials"
3. Click "Sign In"
4. Start managing your gaming store!
